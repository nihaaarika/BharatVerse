# Project Requirements

## Summary

Goal Detector is an AI-powered, student-friendly goal and career discovery assistant.
It helps users reflect on their interests, goals, and constraints and provides a
personalized roadmap in a supportive and ethical manner.



## Target Users

- Students and early-career learners
- Individuals exploring career direction
- Users seeking clarity without external pressure



## Functional Requirements

The system must:

- Allow users to start without mandatory sign-up
- Collect user input through a guided questionnaire
- Support both multiple-choice and short-text responses
- Analyze interests, goals, and preferences using AI
- Generate a personalized roadmap based on user inputs
- Display encouragement and appreciation messages
- Support back navigation without data leakage
- Regenerate the roadmap on demand



## Non-Functional Requirements

- The interface must be calm, clean, and easy to navigate
- The system should respond within a reasonable time
- Animations and effects must not distract from content
- The application must work on modern web browsers
- The UI should be accessible and readable



## AI Requirements

- The AI must act as a guide, not a decision-maker
- The AI should handle missing or optional data gracefully
- The AI must avoid judgmental or prescriptive language
- Outputs should be clear, human-readable, and supportive
- AI responses should be consistent for similar inputs



## Privacy & Ethics Requirements

- User data must not be permanently stored
- Sensitive questions must be optional
- No personal data should be exposed
- The system should be transparent about data usage



## Constraints & Assumptions

- The project is developed as an MVP
- Roadmaps are advisory, not guarantees
- Internet access is required for AI responses
- The system does not replace professional counseling



## Out of Scope

- Full career placement or job guarantees
- Long-term data storage or analytics
- Advanced psychometric testing
- Real-time mentoring
